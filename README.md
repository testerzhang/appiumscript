## 安装说明

1. 需要在电脑手机搭建Appium环境，这里有一篇很久以前写的安装环境：https://www.jianshu.com/p/828d7a5f59d4，可能会有出入，仅供参考。

   这里就不列举搭建的过程了。



2. 本人使用的是Python 3.8.6，需要安装相关Python第三方库

```
cd 具体的脚本文件夹
pip3 install -r requirements.txt
```



3. 本脚本只适用控制安卓手机。
